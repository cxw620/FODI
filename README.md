# FODI

Fast OneDrive Index / FODI，无需服务器的 OneDrive 快速列表程序

华为云函数工作流FunctionGraph  重制版

#### 预览

- [DEMO(暂未开放)](htttps://www.baidu.com)

#### 功能

- 接近秒速列表(取决于这个静态网页放在哪里)
- 指定展示路径
- 特定文件夹加密
- 无需服务器~~免费部署~~
- 基本文本、图片、音视频和 Office 三件套预览

#### 缺点

- 功能简单，界面还是略简陋
- 不支持巨硬家的 IE 和 UWP 版 EDGE 浏览器
- 可能不支持包含千级数量以上文件的文件夹展示
- 不支持空文件夹
